# 🐍 Advanced Python Assignments

A comprehensive collection of **15 Python assignments** covering fundamental to intermediate programming concepts. This repository demonstrates hands-on practice with Python syntax, control structures, data structures, functions, modules, and essential libraries used in data analysis and visualization.

---

## 📖 About the Repository

This repository was created to strengthen Python programming skills through practical assignments. Each notebook focuses on a specific topic and includes examples, exercises, and solutions to reinforce learning.

The assignments progress from basic Python concepts to powerful libraries such as **NumPy**, **Pandas**, and **Matplotlib**, providing a solid foundation for **Data Analysis**, **Machine Learning**, and **Software Development**.

---

## 🎯 Learning Objectives

- Understand Python syntax and programming fundamentals.
- Develop problem-solving and logical thinking skills.
- Practice writing clean and reusable Python code.
- Learn Python data structures and built-in functions.
- Perform numerical computations using NumPy.
- Manipulate and analyze data using Pandas.
- Create meaningful visualizations with Matplotlib.

---

## 📚 Assignments

| Assignment | Topic | Description |
|------------|-------|-------------|
| Assignment-01 | Variables | Variables, data types, and type casting |
| Assignment-02 | Operators | Arithmetic, comparison, logical, assignment, and bitwise operators |
| Assignment-03 | Conditional Statements | Decision making using if, elif, and else |
| Assignment-04 | While & For Loops | Iteration, nested loops, break, continue, and pass |
| Assignment-05 | Defined Functions | Functions, parameters, arguments, return values, and scope |
| Assignment-06 | Modules | Importing built-in modules and creating custom modules |
| Assignment-07 | Strings | String methods, formatting, slicing, and manipulation |
| Assignment-08 | Lists | List operations, methods, indexing, slicing, and comprehensions |
| Assignment-09 | Tuples | Tuple operations, packing, unpacking, and methods |
| Assignment-10 | Dictionaries | Dictionary creation, methods, nested dictionaries, and iteration |
| Assignment-11 | Lambda Functions | Anonymous functions, map(), filter(), and reduce() |
| Assignment-12 | Datetime | Working with dates, times, parsing, and formatting |
| Assignment-13 | NumPy | Arrays, indexing, slicing, vectorized operations, and mathematical functions |
| Assignment-14 | Pandas | Series, DataFrames, data cleaning, filtering, and analysis |
| Assignment-15 | Matplotlib | Line plots, bar charts, histograms, pie charts, scatter plots, and customization |
